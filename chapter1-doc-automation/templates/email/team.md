# 팀원용 이메일 템플릿

## 형식

```
제목: [주간 보고] {시작일} ~ {종료일} 상세 분석

팀원 여러분 안녕하세요,

금주 실적 상세 분석 결과를 공유합니다.

📊 주간 실적 요약
  - 총 매출 / 판매량 / 일 평균 매출
  - 판매 상품 수 / 분석 기간

🏆 TOP 5 상품
  (순위별 상품명, 매출, 판매량)

📁 카테고리별 매출
  (카테고리별 매출액 및 비중)

📋 다음 주 액션 아이템
  1. 구체적 액션 → 담당자
  2. ...

상세 차트 및 분석은 첨부된 PPT를 참고해주세요.
```

## 톤 가이드

- 상세하고 데이터 중심
- 액션 아이템 명확히 포함
- 담당자 지정 권장
- 해요체 또는 합니다체
- 이모지 적절히 활용하여 가독성 확보
