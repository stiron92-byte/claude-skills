# 임원용 이메일 템플릿

## 형식

```
제목: [주간 보고] {시작일} ~ {종료일} 실적 요약

안녕하세요,

금주 주간 실적을 보고드립니다.

■ 핵심 지표
  - 총 매출: {총매출}원 (전주 대비 {변화율}%)
  - 총 판매량: {총판매량}개
  - 일 평균 매출: {일평균매출}원

■ TOP 3 상품
  1. {상품1} ({매출1}원)
  2. {상품2} ({매출2}원)
  3. {상품3} ({매출3}원)

■ 요약
  {1~2문장 핵심 요약}

상세 분석은 첨부된 PPT 보고서를 참고해주시기 바랍니다.

감사합니다.
```

## 톤 가이드

- 간결하고 핵심 중심
- 숫자와 결론 위주
- 3분 이내에 읽을 수 있는 분량
- 불필요한 수식어 제거
- 하십시오체 사용
